Importar Productos desde Excel

Formato requerido

El archivo debe ser .xlsx con estas columnas en la fila 1:

  • Nombre — obligatorio
  • SKU, Categoría, Descripción
  • Precio Venta, Costo, Stock, Stock Mínimo, Unidad

💡 Descargá la plantilla actual como base para importar.

Si un producto con el mismo SKU o nombre ya existe, se actualizará. El stock solo se asigna a productos nuevos.
